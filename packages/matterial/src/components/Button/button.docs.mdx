---
name: Button
description: Buttons trigger actions when clicked.
---

## Usage

```js
import { Button } from 'matterial'

function ButtonExample() {
  return <Button onClick={() => console.log('I got clicked!')}>Click Me</Button>
}
```

## Fixed Width

`width = number | percent`

Text will overflow or center on fixed-width buttons.

<Container row nowrap>
  <Button width="50%" variant="contained" color="primary">Minim eu exercitation pariatur cillum eiusmod occaecat enim proident ipsum.</Button>
  <Button width="50%" variant="contained" color="warning">Foo</Button>
</Container>
