---
name: Link
description: Hyperlinks between pages.
---

## Usage

```jsx
import { Link } from 'matterial'

function App() {
  return <Link href="#">Anchor</Link>
}
```

## Link Props

### Color

`unstyled = boolean`

## Button Link

`to = string`

Create a button-like link with the `to` prop on a `<Button>` component.

<Container row>
  <Button to="#" color="primary">
    Home
  </Button>
  <Button to="#" variant="contained" color="primary">
    Home
  </Button>
</Container>

```jsx
<Button to="#" color="primary">Home</Button>
<Button to="#" variant="contained" color="primary">Home</Button>
```

## NextLink

Wrap `Link` components in a component imported from `next/link`. **Pass the `passHref` prop to the wrapper to <a href="https://nextjs.org/docs/api-reference/next/link#if-the-child-is-a-custom-component-that-wraps-an-a-tag">ensure the link component renders a `href` tag</a>.**

```jsx
import NextLink from 'next/link'
import { Link } from 'matterial'

const App = () => (
  <NextLink href="/" passHref>
    <Link color="secondary">Home</Link>
  </NextLink>
)
```
